平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。

================================================================================
功能描述
================================================================================
本样例主要展示TMR0的基本计数功能。

说明：
本样例设置TMR0时钟源为XTAL32，使能计数匹配中断，每0.5S中断一次并切换LED_BLUE的亮
灭状态；同时使能按键K5的下降沿产生内部触发事件，该事件可以使定时器停止计数。

================================================================================
测试环境
================================================================================
测试用板:
---------------------
EV_F448_LQ80_Rev1.0

辅助工具:
---------------------
无

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）打开工程编译并全速运行；
2）按下复位按键，观察LED_BLUE以1Hz的频率闪烁；
3）短按按键K5，触发定时器计数停止，LED_BLUE停止闪烁。

================================================================================
注意
================================================================================
在异步时钟下，必须保证所选的异步时钟开启；否则TMR0将不会计数，配置也不一定准确。
异步时钟下，对TMR0寄存器操作需要3个异步时钟才能写入。
